`MarigoldProvider` is the container for all Marigold applications. It defines the theme and other application level settings.
Without the `MarigoldProvider` you can't get the theme on your components. So it is nessesary to use.

<Preview>
  <MarigoldProvider theme={theme}>
    <Button variant="primary">Hello World</Button>
  </MarigoldProvider>
</Preview>

## Usage

### Import

```tsx
import { MarigoldProvider } from '@marigold/components';
```
