The `Label` component is used to accosiate and describe a form field.

You need a `<Label>` for form components like [`<Radio>`](/components/radio/) or [`<Input>`](/components/input/). In some cases like for the [`<Select>`](/components/select/) is a property called `label` which behaves as `<Label>`. Usually you have to add a label also for accessibility reasons.

A `<Label>` can be from type `label` or `span`. For that you have to set the `as` property.

<Preview>
  <Label>Some Label</Label>
</Preview>

## Usage

### Import

To import the component you just have to use this code below.

```tsx
import { Label } from '@marigold/components';
```
