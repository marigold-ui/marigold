With the `<Input>` component you can add a HTML `<input>` field for different types of text to your form.

It is a basic form of the [`<TextField>`](/component/text-field/). It hasn't error messages, help texts or label given. It is just the plain input element.

<Preview>
  <Label htmlFor="input">
    Give me your input:
    <Input id="input" type="text" placeholder="Placeholder" />
  </Label>
</Preview>

## Usage

### Import

```tsx
import { Input } from '@marigold/components';
```
