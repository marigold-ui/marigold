Breakout is a Layout element which lets you span the full row in a CSS grid context e.g. our Container component. You can use `horizontalAlign` or `verticalAlign` to place the items inside the Breakout element.

```tsx preview file=components/Breakout/basic-breakout.demo.tsx

```

## Props

<PropsTable
  props={[
    {
      property: 'horizontalAlign',
      type: '"top", "bottom", "center"',
      description:
        'Horizontal alignment of the items inside the Breakout element',
      default: '"center"',
    },
    {
      property: 'verticalAlign',
      type: '"left", "center", "right"',
      description:
        'Vertical alignment of the items inside the Breakout element',
      default: '"center"',
    },
  ]}
/>
