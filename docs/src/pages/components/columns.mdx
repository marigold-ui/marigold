Columns component with sized columns in one row. Add an array of numbers to set the size of the children. The columns array length and the count of children must be the same. You can add space between the columns und set a collapseAt prop to collapse the columns at a certain width.

```tsx preview file=components/Columns/basic-columns.demo.tsx

```

## Props

<PropsTable
  props={[
    {
      property: 'columns (required)',
      description:
        'An array of numbers to set the size of the children. The columns array length and the count of children must be the same.',
      type: 'number[]',
      default: '',
    },
    {
      property: 'space',
      description: 'Space between the columns.',
      type: 'ResponsiveStyleValue<string>',
      default: 'none',
    },
    {
      property: 'collapseAt',
      description: 'Collapse the columns at a certain width.',
      type: 'string',
      default: '40em',
    },
  ]}
/>
