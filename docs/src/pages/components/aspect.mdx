The Aspect component is used to embed responsive images, videos, maps, etc. You can use a predefined ratio and maximum width.

<Preview>
  <BasicAside />
</Preview>

## Props

<PropsTable
  props={[
    {
      property: 'ratio',
      type: 'square, landscape, portrait, widescreen, ultrawide, golden',
      description: 'The aspect ratio of the image.',
      default: 'square',
    },
    {
      property: 'maxWidth',
      type: 'string',
      description: 'The maximum width of the image.',
      default: '100%',
    },
  ]}
></PropsTable>
