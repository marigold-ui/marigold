The `<Aside>` is a responsive layout component. It should be used if you want to layout side content next to the main content. The side has a fixed with and the other one(the main content) can have a dynamic width.

The `<Aside>` breaks automatically when the screen gets too small for the content.

It is important to note that the `<Aside>` must have exactly too children, where none is a `<Fragment>`.

## Props

<PropsTable
  props={[
    {
      property: 'space',
      type: 'ResponsiveStyleValue<string>',
      description: 'The space between the content and the aside.',
      default: 'none',

    {
      property: 'side',
      type: '"left" | "right"',
      description: 'The side of the content.',
      default: 'left',
    },
    {
      property: 'sideWidth',
      type: 'ResponsiveStyleValue<string>',
      description: 'The side with of the content.',
      default: 'none',
    },
    {
      property: 'wrap',
      type: 'Non-zero Percentage',
      description: 'The wrap of the content.',
      default: '50%',
    },
    {
      property: 'strech',
      type: 'boolean',
      description: 'strech the aside.',
      default: 'true',
    },
  ]}
></PropsTable>
