The `<Header>` is a content component, like the `<Body>` and `<Footer>` and is used to help structure the contents of components.

It's used like the HTML `<header>` tag.

The `<Header>` component represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements.

For additional information please read the [MDN Documentation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/header) or [HMTL Living Standard Doucmentation](https://html.spec.whatwg.org/multipage/sections.html#the-header-element)
