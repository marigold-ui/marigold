The `<Box>` is a basic component like the HTML div element. You can modify the component with different style props. It is also possible to write `__baseCSS` and `css` styles on the `<Box>`.

You can use the `as` prop to let the `<Box>` behave as any Marigold component or HTML Element with all its properties.

For a more detailed description, see [Box Primitive](../../foundation/box-primitive).

<Preview>
  <Box border="1px solid gray" p="small" borderRadius="5px">

</Preview>

## Usage

### Import

To import the component you just have to use this code below.

```tsx
import { Box } from '@marigold/components';
```
