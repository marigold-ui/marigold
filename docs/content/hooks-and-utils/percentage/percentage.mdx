---
title: Percentage
caption: Helper used to format numeric percentage values.
---

`<Percentage>` helper component is used to formate numbers as a percentage value based on the current language .

## Usage

### Import

To import the component you just have to use this code below.

```tsx
import { Percentage } from '@marigold/system';
```
