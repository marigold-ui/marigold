---
title: Numeric
caption: Helper used to format numeric values.
---

`<Numeric>` helper component is used to format numbers based on the current language, this component is built based on [Intl.NumberFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat).

## Usage

### Import

To import the component you just have to use this code below.

```tsx
import { Numeric } from '@marigold/system';
```
