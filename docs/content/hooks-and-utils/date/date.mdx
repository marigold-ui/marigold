title: DateFormatter
caption: Helper used to format date values.
---

`<DateFormatter>` helper component is used to formate date values based on the current language .

## Usage

### Import

To import the component you just have to use this code below.

```tsx
import { DateFormatter } from '@marigold/system';
```
