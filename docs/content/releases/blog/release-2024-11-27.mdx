## Improvements

### Components

#### Slider

Along with a complete overhaul of the documentation and stories for our `<Slider>`, the component has received several visual and functional improvements.

We've introduced a visible indicator for the selected track to enhance usability. and the `<Slider>` now supports dual thumbs, allowing users to select a range seamlessly. To make the `<Slider>` even more versatile, it now functions as a standard form component, complete with support for a `name` attribute and an optional help text.
