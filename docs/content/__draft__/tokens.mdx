=== COLOR TOKENS ===

## Structure

- **base:** default values for a certain category of tokens
- **inverted:** alternative default values when the base value does not have a good constrast ratio
- **brand:** company's primary brand color
- **accent:** company's accent brand color (Schmuckfarbe)
- **error:** feedback color when there is an error or a destructive action
- **success:** feedback color when something was successful
- **warning:** feedback color when the users attention is required
- **info:** feedback color when there is some auxiliary information

### additionally, only for background

- **surface:** colors for elevation levels

## Background Tokens

### Base

- **hover:** used when something is hovered
- **active:** default active color (e.g. button pressed)
- **disabled:** used when something is disabled

### Selected

- **DEFAULT:** used as default selected
- **input:** used for input that need a more apparent visual appearance (raio, checkbox, switch)

### Surface (https://atlassian.design/foundations/elevation)

- **DEFAULT:** used for things like the `<body>` element
- **raised:** raised surfaces, like cards
- **overlay:** surfaces that are very high! like popover
