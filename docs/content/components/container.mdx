import { Box, Container, Text } from '@marigold/components';

# Container

## Description

Containers are used to constrain a content's width to the current breakpoint, while keeping it fluid. You can use `header`or `content`as content type and set the `maxWidth` to `small`, `medium` or `large`.

## Usage

```tsx onlyCode
import { Container } from '@marigold/components';
```

<Box
  as={Container}
  __baseCSS={{ border: '1px dashed gray', borderBottom: 'none' }}
  p="xsmall"
  contentType="header"
>
  <Text as="h5" variant="headline5">
    Medium sized container header
  </Text>
</Box>
<Box as={Container} border="1px dashed gray" p="xsmall" height="180px">
  <Text>Medium sized container content</Text>
</Box>

## Properties (API)

| Property      | Type                   | Default   |
| :------------ | :--------------------- | :-------- |
| `contentType` | `content, header`      | `content` |
| `size`        | `small, medium, large` | `medium`  |
| `align`       | `left, center, right`  | `left`    |

## Variants

### Small header

```tsx
<Container contentType="header" size="small">
  <Text as="h2" variant="headline2">
    Star Wars - The Empire Strikes Back
  </Text>
</Container>
```

### Medium header

```tsx
<Container contentType="header">
  <Text as="h2" variant="headline2">
    Star Wars - The Empire Strikes Back
  </Text>
</Container>
```

### Large header

```tsx
<Container contentType="header" size="large">
  <Text as="h2" variant="headline2">
    Star Wars - The Empire Strikes Back
  </Text>
</Container>
```

### Small content

```tsx
<Container size="small">
  <Text>
    It is a dark time for the Rebellion. Although the Death Star has been
    destroyed, Imperial troops have driven the Rebel forces from their hidden
    base and pursued them across the galaxy.
  </Text>
</Container>
```

### Medium content

```tsx
<Container>
  <Text>
    Evading the dreaded Imperial Starfleet, a group of freedom fighters led by
    Luke Skywalker has established a new secret base on the remote ice world of
    Hoth.
  </Text>
</Container>
```

### Large content

```tsx
<Container size="large">
  <Text>
    The evil lord Darth Vader, obsessed with finding young Skywalker, has
    dispatched thousands of remote probes into the far reaches of space....
  </Text>
</Container>
```

## Align content center

In some cases, the width of the content can be smaller than the container's width. You can use the align prop to align the content left, center or right.

```tsx
<Box as={Container} bg="blue10" align="center" p="small">
  <Text as="h2" variant="headline2">
    Star Wars - The Empire Strikes Back
  </Text>
  <Box pt="xsmall" width="100%">
    <Text>
      It is a dark time for the Rebellion. Although the Death Star has been
      destroyed, Imperial troops have driven the Rebel forces from their hidden
      base and pursued them across the galaxy. Evading the dreaded Imperial
      Starfleet, a group of freedom fighters led by Luke Skywalker has
      established a new secret base on the remote ice world of Hoth. The evil
      lord Darth Vader, obsessed with finding young Skywalker, has dispatched
      thousands of remote probes into the far reaches of space....
    </Text>
  </Box>
  <Box width="20ch">
    <ul>
      <li>Luke</li>
      <li>Leia</li>
      <li>Han Solo</li>
      <li>Chewbacca</li>
      <li>R2D2</li>
      <li>C3PO</li>
      <li>Darth Vader</li>
    </ul>
  </Box>
</Box>
```
