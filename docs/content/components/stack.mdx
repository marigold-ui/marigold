# Stack

Layout component to stack elements. This component uses the spaces from the given theme.

<Box border="1px solid #ced4da" bg="#e9ecef" p="small">
  <Stack space="small">
    <Headline level="2">Header</Headline>
    <Text>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Text>
    <Text>
      Phasellus ipsum tortor, aliquet dapibus fermentum in, mollis vel metus.
    </Text>
  </Stack>
</Box>

```tsx onlyCode
import { Stack } from '@marigold/components';
```

## Props

| Property | Type                           | Default |
| :------- | :----------------------------- | :------ |
| `space`  | `ResponsiveStyleValue<string>` | `none`  |
| `align`  | `left, right, center`          | `left`  |

## Examples

### Spacing

```tsx
<Stack space="large">
  <Headline level="2">Header</Headline>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse
    dignissim dapibus elit, vel egestas felis pharetra non. Cras malesuada,
    massa nec ultricies efficitur, lectus ante consequat magna, a porttitor
    massa ex ut quam.
  </Text>
  <Text>
    Phasellus ipsum tortor, aliquet dapibus fermentum in, mollis vel metus.
    Vestibulum malesuada ante eu velit malesuada, nec ultricies sapien finibus.
    Aenean rutrum in sem a ullamcorper. Integer ut euismod urna. Interdum et
    malesuada fames ac ante ipsum primis in faucibus.
  </Text>
</Stack>
```

### Nested

```tsx
<Stack space="large">
  <Stack space="medium">
    <Headline level="2">Stack Content #1</Headline>
    <Text>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Text>
    <Text>
      Phasellus ipsum tortor, aliquet dapibus fermentum in, mollis vel metus.
    </Text>
  </Stack>
  <Stack space="medium">
    <Headline level="2">Stack Content #2</Headline>
    <Text>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Text>
    <Text>
      Phasellus ipsum tortor, aliquet dapibus fermentum in, mollis vel metus.
    </Text>
  </Stack>
</Stack>
```

### Alignment

```tsx
<>
  <Stack space="medium">
    <Text>Left</Text>
  </Stack>
  <Stack space="medium" align="center">
    <Text>Center</Text>
  </Stack>
  <Stack space="medium" align="right">
    <Text>Right</Text>
  </Stack>
</>
```
