The `Grid` component is a versatile tool to arrange child elements within a two-dimensional grid structure ([CSS Grid](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout/Basic_Concepts_of_Grid_Layout)). It allows full control over columns and rows sizes, spacing between children, and the overall height. However, due to its extensive customization options, it's best suited for situations where simpler layout components like Tiles or Columns aren't sufficient.

## Import

```tsx
import { Grid } from '@marigold/components';
```
