Columns component with sized columns in one row. Add an array of numbers to set the size of the children. The columns array length and the count of children must be the same. You can add space between the columns und set a collapseAt prop to collapse the columns at a certain width.

<Columns space="xsmall" columns={[2, 8, 2]}>
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
</Columns>

```tsx onlyCode
import { Columns } from '@marigold/components';
```

## Props

| Property        | Type                           | Default  |
| :-------------- | :----------------------------- | :------- |
| `columns` (req) | `number[]`                     |          |
| `space`         | `ResponsiveStyleValue<string>` | `none`   |
| `collapseAt`    | `40em`                         | `string` |

## Examples

### Columns and a space between them

```tsx expandCode
<Columns space="xsmall" columns={[2, 8, 2]}>
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
</Columns>
```

### Columns collapsing at a certain width

```tsx
<Columns collapseAt="50em" space="xsmall" columns={[2, 10]}>
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
</Columns>
```

### Page layout

Create a page layout with three columns

<Columns space="xsmall" columns={[2, 8, 2]}>
  <Box border="1px solid #ced4da" bg="#e9ecef" height="250px" p="xsmall">
    Left Sidebar
  </Box>
  <Box border="1px solid #ced4da" bg="#e9ecef" height="250px" p="xsmall">
    Main Content
  </Box>
  <Box border="1px solid #ced4da" bg="#e9ecef" height="250px" p="xsmall">
    Right Sidebar
  </Box>
</Columns>
