# Radio

## Description

With the Radio Button component you can add a HTML `<input>` element with `type="radio"` to your form.
The variant can be added with the variant prop. The default variant is `default`.

## Properties

| Property                  | Type      | Default   |
| :------------------------ | :-------- | :-------- |
| `id`                      | `string`  |           |
| `variant` (optional)      | `string`  | `default` |
| `label` (optional)        | `string`  |           |
| `required` (optional)     | `boolean` | `false`   |
| `disabled` (optional)     | `boolean` | `false`   |
| `error` (optional)        | `boolean` | `false`   |
| `errorMessage` (optional) | `string`  |           |

## Import

```tsx onlyCode
import { Radio } from '@marigold/components';
```

## Live-Code Example

```tsx expandCode
<Radio checked id="live" value="live" label="Radio label" />
```

## Usage

### Radio standard labeled

```tsx
() => {
  const [state, setState] = React.useState('Mastercard');
  const onChange = changeEvent => {
    setState(changeEvent.currentTarget.value);
  };
  return (
    <>
      <Radio
        id="mc"
        name="payment"
        onChange={onChange}
        checked={state === 'Mastercard'}
        value="Mastercard"
        label="Mastercard"
      />
      <br />
      <Radio
        id="vi"
        name="payment"
        onChange={onChange}
        checked={state === 'Visa'}
        value="Visa"
        label="Visa"
      />
      <br />
      <Radio
        id="ae"
        name="payment"
        onChange={onChange}
        checked={state === 'AmericanExpress'}
        value="AmericanExpress"
        label="AmericanExpress"
      />
    </>
  );
};
```

### Radio disabled

```tsx
<>
  <Radio label="Disabled" id="disabled" disabled />
  <br />
  <Radio label="Checked and disabled" id="checkedDisabled" checked disabled />
</>
```

### Radio with required label

```tsx
<>
  <Radio
    checked
    id="required"
    value="required"
    required
    label="This label is required"
  />
</>
```

### Radio with error and errorMessage

Only use the error and errorMesssage prop together

```tsx
<Radio
  id="error"
  value="error"
  label="This radio has an error"
  error
  errorMessage="radio error"
/>
```
