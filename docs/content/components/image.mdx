# Image

## Description

With the Image component you can render an image.
The variant can be added with the variant prop. The default variant is `fullWidth`.

## Properties

| Property             | Type     | Default     |
| :------------------- | :------- | :---------- |
| `variant` (optional) | `string` | `fullWidth` |

## Import

```tsx onlyCode
import { Image } from '@marigold/components';
```

## Usage

```tsx expandCode
<Image
  src="https://images.unsplash.com/photo-1603910234616-3b5f4a6be2b4?ixlib=rb-1.2.1&ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&auto=format&fit=crop&w=687&q=80"
  alt="event_image"
/>
```
