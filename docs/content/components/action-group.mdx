# ActionGroup

## Description

The ActionGroup component adds two or more components together to form a common group.

## Properties

| Property                       | Type                                  | Default   |
| :----------------------------- | :------------------------------------ | :-------- |
| `variant` (optional)           | `string`                              | `default` |
| `space` (optional)             | `ResponsiveStyleValue<numberâŽ®string>` | `none`    |
| `verticalAlignment` (optional) | `boolean`                             | `false`   |

## Import

```tsx code
import { ActionGroup } from '@marigold/components';
```

## Live-Code Example

```tsx live
<ActionGroup space="8px">
  <Button variant="text" size="small">
    Cancel
  </Button>
  <Button variant="primary" size="small">
    Confirm
  </Button>
</ActionGroup>
```

## Usage

```tsx
<ActionGroup>
  <Button variant="text" size="small">
    Cancel
  </Button>
  <Button variant="secondary" size="small">
    Delete <Delete />
  </Button>
</ActionGroup>
```

### With space

```tsx
<ActionGroup space={8}>
  <Button variant="secondary" size="small">
    <Play />
  </Button>
  <Button variant="secondary" size="small">
    <Pause />
  </Button>
  <Button variant="secondary" size="small">
    <Stop />
  </Button>
</ActionGroup>
```

### With vertical alignment

```tsx
<ActionGroup space={8} verticalAlignment>
  <Button variant="secondary" size="small">
    <Play />
  </Button>
  <Button variant="secondary" size="small">
    <Pause />
  </Button>
  <Button variant="secondary" size="small">
    <Stop />
  </Button>
</ActionGroup>
```
