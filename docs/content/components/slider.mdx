# Slider

With the Slider component you can add a HTML `<input>` element with `type="range"` to your form.
The variant can be added with the variant prop. The default variant ist `default`.

<MarigoldTheme>
  <Label htmlFor="slider">
    Slider:
    <br />
    <Slider id="slider" name="slider" min="0" max="50" />
  </Label>
</MarigoldTheme>

```tsx onlyCode
import { Slider } from '@marigold/components';
```

## Props

| Property             | Type     | Default     |
| :------------------- | :------- | :---------- |
| `variant` (optional) | `string` | `__default` |

## Examples

### Slider with steps

```tsx
<Slider label="Slider with steps" id="slider2" step="25" />
```

### Slider disabled

```tsx
<Slider label="Disabled" id="slider3" disabled />
```

### Slider with required label

```tsx
<Slider label="Required" id="slider4" required />
```
