# Menu

## Description

With the Menu component you can add a Menu Button with Text or Icon as Label and as children some MenuItem components.
The variant can be added with the variant prop. The default variant is `menu`.

## Properties

| Property             | Type       | Default |
| :------------------- | :--------- | :------ |
| `variant` (optional) | `string`   | `menu`  |
| `label` (optional)   | `string`   | `Menu`  |
| `onClick`            | `function` |         |
| `show` (optional)    | `boolean`  | `false` |

## Import

```tsx code
import { Menu } from '@marigold/components';
```

## Live-Code Example

```tsx live
() => {
  const [showMenu, setShowMenu] = React.useState(false);
  const toggle = () => {
    setShowMenu(!showMenu);
  };
  return (
    <Menu onClick={toggle} show={showMenu} label="Menu">
      <MenuItem href="#">Item One</MenuItem>
      <MenuItem href="#">Item Two</MenuItem>
      <MenuItem href="#">...</MenuItem>
    </Menu>
  );
};
```
