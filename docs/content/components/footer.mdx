## Description

The `Footer` component is used to help structure contents of components.

<MarigoldTheme>
  <Footer>This is a footer!</Footer>
</MarigoldTheme>

```tsx onlyCode
import { Footer } from '@marigold/components';
```

## Props

| Name      | Type     | Default | Description                          |
| :-------- | :------- | :------ | :----------------------------------- |
| `variant` | `string` |         | Use a different _variant_ from theme |
| `size`    | `string` |         | Use a different _size_ from theme    |

## Examples

```tsx
<Footer>This is a footer!</Footer>
```
