## Description

The `Header` component is used to help structure content components. It's used like the HTML `<header>` tag.

<MarigoldTheme>
  <Header>This is a Header!</Header>
</MarigoldTheme>

```tsx onlyCode
import { Header } from '@marigold/components';
```

## Props

| Name      | Type     | Default | Description                          |
| :-------- | :------- | :------ | :----------------------------------- |
| `variant` | `string` |         | Use a different _variant_ from theme |
| `size`    | `string` |         | Use a different _size_ from theme    |

## Examples

```tsx
<Header>
  <Headline level="3">Header!</Headline>
  <Text>awesome</Text>
</Header>
```
