# Alert

## Demo

<MarigoldTheme>
  <Alert variant="success">
    <Text>Success message</Text>
  </Alert>
</MarigoldTheme>

## Description

Warnings highlight relevant information related to the page, they should be as close as possible to the related content.
Predefined variants are info, success, danger and warning.

## Properties

| Property             | Type                            | Default   |
| :------------------- | :------------------------------ | :-------- |
| `variant` (optional) | `success, info, error, warning` | `success` |

## Import

```tsx onlyCode
import { Alert } from '@marigold/components';
```

## Usage

<Stack space="small">
  <Box
    display="grid"
    css={{
      gridGap: 'small',
      gridTemplateColumns: 'repeat(auto-fit, minmax(min(500px, 100%), 1fr))',
    }}
  >
    <DoAndDont
      preview={
        <Alert>
          <Inline space="small">
            <Text>Success message</Text>
            <SortDown />
          </Inline>
        </Alert>
      }
    >
      Group several alerts messages.
    </DoAndDont>
    <DoAndDont
      dont
      preview={
        <Stack space="small">
          <Alert variant="error">
            <Text>Error message 1</Text>
          </Alert>
          <Alert variant="error">
            <Text>Error message 2</Text>
          </Alert>
          <Alert variant="error">
            <Text>Error message 3</Text>
          </Alert>
          <Alert variant="error">
            <Text>Error message 4</Text>
          </Alert>
        </Stack>
      }
    >
      Do not show too many alert messages at once.
    </DoAndDont>
  </Box>
</Stack>
