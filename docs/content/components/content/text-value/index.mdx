The `<TextValue>` component is a small inline label designed for selection items such as `<Select.Option>`, `<Menu.Item>`, `<ListBox.Item>`, and `<ComboBox.Item>`. By marking the label explicitly, screen readers announce the right text and keyboard typeahead lands on the right item without any extra wiring.

It has no visual props of its own. Its appearance comes from the surrounding selection item, which keeps the look consistent across menus, selects, and lists. Pair it with [`<Description>`](/components/content/description) when the item also needs a help line.

## Usage

### Inside selection items

The primary use case. Drop `<TextValue>` into a `<Menu.Item>`, `<Select.Option>`, `<ListBox.Item>`, or `<ComboBox.Item>`. The selection item picks it up as its accessible name, so screen reader announcements and keyboard typeahead all line up with the visible label.

## Use inside a selection item

The selection item declares the label region and the right styling for it. Consumers just write the label text inside `<TextValue>` and add a `<Description>` next to it when needed.
