The `TextArea` component allows the user to enter multiline text with a keyboard. It consists of a label, a textarea and a help text. The later also includes error messages.

<Preview>
  <TextArea
    label="Enter a story"
    description="Anything you want to tell me!?"
  />
</Preview>

```tsx
import { TextArea } from '@marigold/components';
```
