# Label

With the Label component you can add a HTML `<Label>` element to your form.
The variant can be added with the variant prop. The default variant is `above`.

<MarigoldTheme>
  <Label htmlFor="labelId">Just a simple label</Label>
</MarigoldTheme>

```tsx onlyCode
import { Label } from '@marigold/components';
```

## Props

| Property              | Type                           | Default |
| :-------------------- | :----------------------------- | :------ |
| `variant` (optional)  | `string`                       | `above` |
| `required` (optional) | `boolean`                      | `false` |
| `color` (optional)    | `ResponsiveStyleValue<string>` | `text`  |
| `htmlFor`             | `string`                       |         |

## Examples

### Label required

```tsx
<Label htmlFor="labelId" required>
  I am a required Label!
</Label>
```

### Label colored

```tsx
<Label htmlFor="labelId" color="info">
  I am a disabled Label!
</Label>
```

### Label above

```tsx
<>
  <Label htmlFor="labelId">Name:</Label>
  <Input id="labelId" />
</>
```

### Label inline

```tsx
<Inline space="xxsmall">
  <Checkbox id="labelId" />
  <Label variant="inline" htmlFor="labelId">
    Agree to Marigold terms
  </Label>
</Inline>
```
