# Label

## Description

With the Label component you can add a HTML `<Label>` element to your form.
The variant can be added with the variant prop. The default variant is `above`.

## Properties

| Property              | Type      | Default |
| :-------------------- | :-------- | :------ |
| `variant` (optional)  | `string`  | `above` |
| `required` (optional) | `boolean` | `false` |
| `htmlFor`             | `string`  |         |

## Import

```tsx code
import { Label } from '@marigold/components';
```

## Live-Code Example

```tsx live
<Label htmlFor="labelId">Just a boring label</Label>
```

## Usage

### Label required

```tsx
<Label variant="inline" htmlFor="labelId" required>
  I am a required Label!
</Label>
```

### Label above

```tsx
<Label htmlFor="labelId">I am a Label!</Label>
```

### Label inline

```tsx
<Label variant="inline" htmlFor="labelId">
  I am a Label!
</Label>
```

### Label disabled

```tsx
<Label variant="disabled" htmlFor="labelId">
  I am a disabled Label!
</Label>
```

### Collective Label

```tsx
<Label htmlFor="labelId">
  Collective Label
  <Box display="flex" justifyContent="space-between" width="25%">
    <Radio id="one" name="collective" value="RadioOne" label="Radio One" />
    <Radio id="two" name="collective" value="RadioTwo" label="Radio Two" />
  </Box>
</Label>
```

### Section Label

```tsx
<Label htmlFor="labelId" variant="section">
  Section Label
  <Divider variant="section" />
</Label>
```

### Label Examples

```tsx
<Label htmlFor="labelId" variant="section">
  Radios
  <Divider variant="section" />
  <Label htmlFor="labelId">
    Radio Group
    <Box display="flex" justifyContent="space-between" width="50%">
      <Radio id="one" name="collective" value="RadioOne" label="Radio One" />
      <Radio id="two" name="collective" value="RadioTwo" label="Radio Two" />
      <Radio
        id="three"
        name="collective"
        value="RadioThree"
        label="Radio Three"
      />
    </Box>
  </Label>
</Label>
```

```tsx
<Label htmlFor="labelId" variant="section">
  Dropdowns
  <Divider variant="section" />
  <Box display="flex" justifyContent="space-between" width="50%">
    <Select
      label="Dropdown One"
      placeholder="Select Primary Color"
      htmlFor="id"
    >
      <Item key="Red">Red</Item>
      <Item key="Orange">Orange</Item>
    </Select>
    <Select
      label="Dropdown Two"
      placeholder="Select Secondary Color"
      htmlFor="id2"
    >
      <Item key="Red">Red</Item>
      <Item key="Orange">Orange</Item>
    </Select>
  </Box>
</Label>
```
