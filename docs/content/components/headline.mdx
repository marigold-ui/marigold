## Props

<!-- list all props with values and description -->

| Name      | Type     | Default     | Description                                               |
| :-------- | :------- | :---------- | :-------------------------------------------------------- |
| `variant` | `string` |             | Use a different _variant_ from theme                      |
| `size`    | `string` | `'level-1'` | Use a different _size_ from theme                         |
| `level`   | `string` | `1`         | set a different _level_ from theme, values are from 1 - 6 |
