# Center

## Description

Center is a layout component that centers its children within itself.

## Demo

<MarigoldTheme>
  <Center maxHeight="large" maxWidth="huge">
    <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
  </Center>
</MarigoldTheme>

## Properties (API)

| Property         | Type                           | Default  |
| :--------------- | :----------------------------- | :------- |
| `maxHeight`      | `string`                       |          |
| `maxWidth`       | `string`                       |          |
| `textAlign`      | `left, center, right`          | `center` |
| `space`          | `ResponsiveStyleValue<string>` | `none`   |
| `superCenntered` | `boolean`                      | `false`  |

## Import

```tsx onlyCode
import { Center } from '@marigold/components';
```

## Variants

### Center maxHeight

```tsx expandCode
<Center maxHeight="large">
  <Box border="1px solid #ced4da" bg="#e9ecef" />
</Center>
```

### Center maxWidth

```tsx
<Center maxWidth="xxlarge">
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" />
</Center>
```

### Center textAlign

```tsx
<Center textAlign="left" maxWidth="huge">
  <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" p="small">
    Left aligned Box
  </Box>
</Center>
```
