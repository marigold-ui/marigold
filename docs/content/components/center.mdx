# Center

## Description

Center is a layout component that centers its children within itself.

## Usage

```tsx onlyCode
import { Center } from '@marigold/components';
```

<MarigoldTheme>
  <Center maxHeight="large" maxWidth="huge">
    <Box height="100%" border="1px dashed gray" p="small">
      Centered Box
    </Box>
  </Center>
</MarigoldTheme>

## Properties (API)

| Property    | Type                           | Default  |
| :---------- | :----------------------------- | :------- |
| `maxHeight` | `ResponsiveStyleValue<string>` |          |
| `maxWidth`  | `ResponsiveStyleValue<string>` |          |
| `textAlign` | `left, center, right`          | `center` |

## Variants

### Center maxHeight

```tsx expandCode
<Center maxHeight="large">
  <Box height="100%" border="1px dashed gray" p="small">
    Centered Box
  </Box>
</Center>
```

### Center maxWidth

```tsx
<Center maxWidth="xxlarge">
  <Box height="100%" border="1px dashed gray" p="small">
    Centered Box
  </Box>
</Center>
```

### Center textAlign

```tsx
<Center textAlign="left" maxWidth="huge">
  <Box height="100%" border="1px dashed gray" p="small">
    Left aligned Box
  </Box>
</Center>
```
