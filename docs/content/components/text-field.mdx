The `TextField` component allows user so enter text with a keyboard. It consists of a label, an input field and a help text. The later also includes error messages.

<Preview>
  <TextField label="Try me" description="You should really try this!" />
</Preview>

```tsx onlyCode
import { TextField } from '@marigold/components';
```

## Examples

### Simple Text Field

```tsx
<TextField label="Firstname" />
```

### Disabled Text Field

```tsx
<TextField label="Field disabled" disabled />
```

### Required Text Field

```tsx
<TextField label="Name" required />
```

### Text Field with an Error

```tsx
<TextField label="Name" error errorMessage="Something went wrong!" />
```

### Not every number is a `number`!

[Just because the input contains numbers, it is not necessarily correctly to use `input[type=number]`!](https://www.coywolf.news/webmaster/accessible-number-input/)

```tsx
const FieldTextErrorNumberImput = () => {
  const [value, setValue] = React.useState('');
  const error = value.length > 0 && !/^\d+$/.test(value);

  return (
    <TextField
      label="Promo Code"
      description="You'll find your promo code on the back of your ticket."
      errorMessage="Your promo code is not valid! Please review your input."
      value={value}
      onChange={setValue}
      error={error}
      inputmode="numeric"
      pattern="[0-9]*"
    />
  );
};
```
