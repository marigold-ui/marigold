The card component can be used for organizing information inside the UI, as well it provides a better readability and helps the user to find information in an easier way.

<MarigoldTheme>
  <Card>
    <Text>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse
      dignissim dapibus elit, vel.
    </Text>
    <Link href="#">
      <Text>Read more...</Text>
    </Link>
  </Card>
</MarigoldTheme>

```tsx onlyCode
import { Card } from '@marigold/components';
```

## Props

| Property  | Type     | Default | Description                          |
| :-------- | :------- | :------ | :----------------------------------- |
| `variant` | `string` |         | Use a different _variant_ from theme |
| `size`    | `string` |         | Use a different _size_ from theme    |

## Examples

### Standard

```tsx expandCode
<Card>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse
    dignissim dapibus elit, vel.
  </Text>
  <Link href="#">
    <Text>Read more...</Text>
  </Link>
</Card>
```

### Card with form fields

```tsx
<Card>
  <TextField label="Firstname" />
  <TextField label="Lastname" />
</Card>
```

### Card with select

```tsx
<Card>
  <Select label="Favorite Color" placeholder="Choose a color" width={450}>
    <Item>Red</Item>
    <Item>Orange</Item>
    <Item>Yellow</Item>
    <Item>Green</Item>
    <Item>Blue</Item>
    <Item>Purple</Item>
  </Select>
</Card>
```
