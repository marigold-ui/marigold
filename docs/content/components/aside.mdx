The `<Aside>` layout is useful when you want to set two things next to each other and the one has a fixed width (the aside) and the other can have a dynamic width (the content).

It is important to note that the `<Aside>` must have exactly too children, where none is a `<Fragment>`.

</Preview>

## Props

<PropsTable
  props={[
    {
      property: 'space',
      type: 'ResponsiveStyleValue<string>',
      description: 'The space between the content and the aside.',
      default: 'none',
    },
    {
      property: 'side',
      type: '"left" | "right"',
      description: 'The side of the content.',
      default: 'left',
    },
    {
      property: 'sideWidth',
      type: 'ResponsiveStyleValue<string>',
      description: 'The side with of the content.',
      default: 'none',
    },
    {
      property: 'wrap',
      type: 'Non-zero Percentage',
      description: 'The wrap of the content.',
      default: '50%',
    },
    {
      property: 'strech',
      type: 'boolean',
      description: 'strech the aside.',
      default: 'true',
    },
  ]}
></PropsTable>
