# Textarea

## Description

The Textarea component consists of an `<textarea>` element and a `<label>`.

## Properties

| Property                  | Type      | Default     |
| :------------------------ | :-------- | :---------- |
| `variant` (optional)      | `string`  | `__default` |
| `label`                   | `string`  |             |
| `htmlFor`                 | `string`  |             |
| `error` (optional)        | `boolean` |             |
| `errorMessage` (optional) | `string`  |             |
| `required` (optional)     | `boolean` |             |

## Import

```tsx onlyCode
import { Textarea } from '@marigold/components';
```

## Live-Code Example

```tsx expandCode
<Textarea
  label="Textarea One"
  htmlFor="textareaOne"
  placeholder="Place your text here.."
/>
```

## Usage

### Basic Textarea

```tsx
<Textarea
  label="Textarea Two"
  htmlFor="textareaTwo"
  placeholder="Placeholder"
/>
```

### Disabled Textarea

```tsx
<Textarea
  label="Textarea Three"
  htmlFor="textareaThree"
  placeholder="Placeholder disabled"
  disabled
/>
```

### Required Textarea

```tsx
<Textarea label="Required Textarea" htmlFor="required" required />
```

### Error Textarea

```tsx
<Textarea
  label="Required Textarea with error"
  htmlFor="error"
  required
  error
  errorMessage="Oh no, we have an error!"
/>
```
