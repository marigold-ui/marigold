# Textarea

## Description

With the Textarea component you can add a HTML `<textarea>` element for larger text input to your form.
The variant can be added with the variant prop. The default variant is `default`.
If you use the label prop you have to add htmlFor. Optionally you can pass an error with errorMessage or the required prop.

## Properties

| Property                  | Type      | Default    |
| :------------------------ | :-------- | :--------- |
| `variant` (optional)      | `string`  | `default`  |
| `label` (optional)        | `string`  |            |
| `htmlFor` (optional)      | `string`  | `textarea` |
| `error` (optional)        | `boolean` |            |
| `errorMessage` (optional) | `string`  |            |
| `required` (optional)     | `boolean` |            |

## Import

```tsx onlyCode
import { Textarea } from '@marigold/components';
```

## Live-Code Example

```tsx expandCode
<Textarea placeholder="Place your text here.." />
```

## Usage

### Basic Textarea

```tsx
<Textarea placeholder="Placeholder" />
```

### Disabled Textarea

```tsx
<Textarea placeholder="Placeholder disabled" disabled />
```

### Labeled Textarea

```tsx
<Textarea label="Labeled" htmlFor="label" />
```

### Required Textarea

```tsx
<Textarea label="Required" htmlFor="required" required />
```

### Error Textarea

```tsx
<Textarea
  label="Required with error"
  htmlFor="error"
  required
  error
  errorMessage="Oh no, we have an error!"
/>
```
