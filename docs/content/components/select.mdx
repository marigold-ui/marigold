# Select

## Description

Dropdown for selecting an item among different options.

## Properties

| Property                  | Type              | Default |
| :------------------------ | :---------------- | :------ |
| `label` (optional)        | `string`          |         |
| `placeholder` (optional)  | `string`          |         |
| `disabled` (optional)     | `boolean`         |         |
| `selectedItem` (optional) | `string`          |         |
| `disabledKeys` (optional) | `array['string']` |         |
| `required` (optional)     | `boolean`         |         |
| `error` (optional)        | `string`          |         |

## Import

```tsx code
import { Select } from '@marigold/components';
```

```tsx code
import { Item } from '@marigold/components';
```

## Live-Code Example

```tsx live
<Select label="Favorite character" placeholder="Select your character">
  <Item>Mario</Item>
  <Item>Luigi</Item>
  <Item>Toad</Item>
  <Item>Yoshi</Item>
  <Item>Bowser</Item>
  <Item>Peach</Item>
</Select>
```

## Usage

### Standard Select

```tsx
<Select label="Favorite Color" placeholder="Choose a color">
  <Item>Red</Item>
  <Item>Orange</Item>
  <Item>Yellow</Item>
  <Item>Green</Item>
  <Item>Blue</Item>
  <Item>Purple</Item>
</Select>
```

### Default selectedItem

```tsx
<Select label="Favorite Color" htmlFor="id" selectedItem="Red">
  <Item key="Red">Red</Item>
  <Item key="Orange">Orange</Item>
  <Item key="Yellow">Yellow</Item>
  <Item key="Green">Green</Item>
  <Item key="Blue">Blue</Item>
  <Item key="Purple">Purple</Item>
</Select>
```

### Disabeld Select

```tsx
<Select label="Disabled Select:" disabled placeholder="disabled Select">
  <Item>disabled Item</Item>
</Select>
```

### Disabled Items

```tsx
<Select label="Favorite Color" htmlFor="id" disabledKeys={['Orange', 'Yellow']}>
  <Item key="Red">Red</Item>
  <Item key="Orange">Orange</Item>
  <Item key="Yellow">Yellow</Item>
  <Item key="Green">Green</Item>
  <Item key="Blue">Blue</Item>
  <Item key="Purple">Purple</Item>
</Select>
```

### Required label Select

```tsx
<Select required label="required Select">
  <Item>Option 1</Item>
  <Item>Option 2</Item>
</Select>
```

### Error Select

```tsx
<Select error="Please choose one option!" label="error Select">
  <Item>Option 1</Item>
  <Item>Option 2</Item>
</Select>
```
