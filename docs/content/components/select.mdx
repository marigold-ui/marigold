# Select

## Description

Dropdown for selecting an item among different options.

## Properties

| Property                 | Type      | Default |
| :----------------------- | :-------- | :------ |
| `label` (optional)       | `string`  |         |
| `placeholder` (optional) | `string`  |         |
| `disabled` (optional)    | `boolean` |         |

## Import

```tsx code
import { Select } from '@marigold/components';
```

```tsx code
import { Item } from '@react-stately/collections';
```

## Live-Code Example

```tsx live
<Select
  label="Favorite character"
  htmlFor="id"
  placeholder="Select your character"
>
  <Item>Mario</Item>
  <Item>Luigi</Item>
  <Item>Toad</Item>
  <Item>Yoshi</Item>
  <Item>Bowser</Item>
  <Item>Peach</Item>
</Select>
```

## Usage

```tsx
<Select label="Favorite Color" htmlFor="id" placeholder="Choose a color">
  <Item>Red</Item>
  <Item>Orange</Item>
  <Item>Yellow</Item>
  <Item>Green</Item>
  <Item>Blue</Item>
  <Item>Purple</Item>
</Select>
```

```tsx
<Select label="Disabled Select:" disabled placeholder="disabled Select">
  <Item>disabled Item</Item>
</Select>
```
