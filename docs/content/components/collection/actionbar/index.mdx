The `<ActionBar>` is a floating toolbar that surfaces bulk actions for the current selection in a collection. When users select one or more rows in a [`<Table>`](../table) (or another collection), the bar slides in, shows the selection count, and offers actions — like edit, delete, or export — that apply to every selected item at once.
