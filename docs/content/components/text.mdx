With the Text component you can easily add different body text HTML elements.

```tsx onlyCode
import { Text } from '@marigold/components';
```

## Props

| Property              | Type                | Default |
| :-------------------- | :------------------ | :------ |
| `variant` (optional)  | `string âŽ® string[]` |         |
| `align` (optional)    | `string`            |         |
| `color` (optional)    | `string`            |         |
| `cursor` (optional)   | `string`            |         |
| `outline` (optional)  | `string`            |         |
| `size` (optional)     | `string`            |         |
| `fontSize` (optional) | `string`            |         |

### Colored Text

```tsx
<Text color="hotpink">Pink is awesome!</Text>
```
