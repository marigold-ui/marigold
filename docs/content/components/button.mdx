# Button

## Description

With the Button component you can render a `button` element.
The style variant and size of the button can be added with the variant prop and is composed like `variant.size`.

## Properties

| Property              | Type                                    | Default   |
| :-------------------- | :-------------------------------------- | :-------- |
| `variant` (optional)  | `primary`, `secondary`, `ghost`, `text` | `primary` |
| `size` (optional)     | `small`, `large`                        | `large`   |
| `disabled` (optional) | `boolean`                               |           |
| `icon` (optional)     | `ReactSVGElement`                       |           |

## Import

```tsx code
import { Button } from '@marigold/components';
```

## Live-Code Example

```tsx live
<Button
  variant="secondary"
  size="small"
  onClick={() => {
    alert('Searching..');
  }}
>
  Search on <Google /> oogle
</Button>
```

## Usage

### Large

```tsx
<Button variant="primary" size="large">
  Primary
</Button>
```

```tsx
<Button variant="secondary" size="large">
  Secondary
</Button>
```

```tsx
<Button variant="ghost" size="large">
  Ghost
</Button>
```

```tsx
<Button variant="text" size="large">
  Text Only
</Button>
```

### Small

```tsx
<Button variant="primary" size="small">
  Primary
</Button>
```

```tsx
<Button variant="secondary" size="small">
  Secondary
</Button>
```

```tsx
<Button variant="ghost" size="small">
  Ghost
</Button>
```

```tsx
<Button variant="text" size="small">
  Text Only
</Button>
```

### Disabled

```tsx
<Button variant="primary" size="large" disabled>
  Primary Large Disabled
</Button>
```

```tsx
<Button variant="secondary" size="large" disabled>
  Secondary Large Disabled
</Button>
```

```tsx
<Button variant="ghost" size="large" disabled>
  Ghost Large Disabled
</Button>
```

```tsx
<Button variant="text" size="large" disabled>
  Text Only Large Disabled
</Button>
```

### Default with icon

```tsx
<Button icon={<Ticket />}>Icon Primary</Button>
```

```tsx
<Button size="small" icon={<Ticket size={20} />}>
  Icon Primary
</Button>
```

```tsx
<Button variant="secondary" icon={<Facebook />}>
  Like me!
</Button>
```

```tsx
<Button variant="secondary" size="small" icon={<Facebook size={20} />}>
  Like me!
</Button>
```

```tsx
<Button variant="ghost" icon={<Id />}>
  Ghost
</Button>
```

```tsx
<Button variant="ghost" size="small" icon={<Id size={20} />}>
  Ghost
</Button>
```

```tsx
<Button variant="text" icon={<FormatSize />}>
  Icon Text only
</Button>
```

```tsx
<Button variant="text" size="small" icon={<FormatSize size={20} />}>
  Icon Text only
</Button>
```

### Large with large icon

```tsx
<Button variant="primary" size="large">
  <Facebook size={48} /> Like me!
</Button>
```

### Small with small icon

```tsx
<Button variant="primary" size="small">
  <Facebook size={16} /> Like me!
</Button>
```

### With colored icon

```tsx
<Button variant="primary" size="small">
  <Facebook fill="black" /> Like me!
</Button>
```
