# ValidationMessage

## Description

Simple colored element to display a validation message as a reaction to an event.

  to={

  }
/>

## Properties

| Property             | Type     | Default |
| :------------------- | :------- | :------ |
| `variant` (optional) | `string` | `error` |

## Import

```tsx onlyCode
import { ValidationMessage } from '@marigold/components';
```

```tsx expandCode
<ValidationMessage>
  <SmilieDissatisfied />
  Error!
</ValidationMessage>
```

## Usage

```tsx
<ValidationMessage>
  <Exclamation />
  Validation message
</ValidationMessage>
```
