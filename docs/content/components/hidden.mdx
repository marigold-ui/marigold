# Hidden

## Description

With the Hidden component you can hide elements or components. E.g. to hide some text in specific cases, for groups or platforms.

## Properties

| Property          | Type      | Default |
| :---------------- | :-------- | :------ |
| `show` (optional) | `boolean` | `false` |

## Import

```tsx onlyCode
import { Hidden } from '@marigold/components';
```

## Usage

```tsx
<>
  <Text>I am visible</Text>
  <Hidden>
    <Text>I am invisible</Text>
  </Hidden>
</>
```

```tsx
() => {
  const [toggleState, setToggleState] = React.useState(false);
  const handleToggleClick = () => {
    setToggleState(!toggleState);
  };
  return (
    <div>
      <Button variant="secondary" size="small" onClick={handleToggleClick}>
        {toggleState ? 'Hide Text' : 'Show Text'}
      </Button>
      <Hidden show={toggleState}>
        <br />
        <Text>I am visible</Text>
      </Hidden>
    </div>
  );
};
```
