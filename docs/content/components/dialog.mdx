# Dialog

## Description

Element to display a Dialog as a reaction to an event. The user style or fill the body of the dialog with content.

## Properties

| Property           | Type       | Default |
| :----------------- | :--------- | :------ |
| `onClose`          | `function` |         |
| `title (optional)` | `string`   |         |

## Import

```tsx code
import { Dialog } from '@marigold/components';
```

## Live-Code Example

```tsx live
() => {
  const { state, openButtonProps, openButtonRef } = useDialogButtonProps();
  return (
    <>
      <Button
        variant="secondary"
        size="small"
        {...openButtonProps}
        ref={openButtonRef}
      >
        Open Dialog
      </Button>
      {state.isOpen && (
        <Dialog title="Dialog Livecode example" {...state}>
          <Text>
            This is your dialog text. You can read some interesting information.
          </Text>
        </Dialog>
      )}
    </>
  );
};
```
