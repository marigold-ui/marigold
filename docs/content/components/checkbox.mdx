# Checkbox

## Description

With the Checkbox component you can add a HTML `<input>` element with `type="checkbox"` to your form.
The variant can be added with the variant prop. The default variant is `__default`.

## Properties

| Property                  | Type      | Default     |
| :------------------------ | :-------- | :---------- |
| `id`                      | `string`  |             |
| `variant` (optional)      | `string`  | `__default` |
| `labelVariant` (optional) | `string`  | `inline`    |
| `required` (optional)     | `boolean` | `false`     |
| `disabled` (optional)     | `boolean` | `false`     |
| `error` (optional)        | `boolean` | `false`     |
| `errorMessage` (optional) | `string`  |             |

## Import

```tsx onlyCode
import { Checkbox } from '@marigold/components';
```

## Live-Code Example

```tsx expandCode
() => {
  const [isChecked, setChecked] = React.useState(false);
  const onChange = () => {
    setChecked(!isChecked);
  };
  return (
    <Checkbox
      id="checkboxLive"
      name="checkboxLive"
      onChange={onChange}
      checked={isChecked}
      value="Checkbox"
    >
      Check the box
    </Checkbox>
  );
};
```

## Usage

### Checkbox standard labeled

```tsx
() => {
  const [isChecked, setChecked] = React.useState(false);
  const onChange = () => {
    setChecked(!isChecked);
  };
  return (
    <Checkbox
      id="myCheckbox"
      name="myCheckbox"
      onChange={onChange}
      checked={isChecked}
      value="Checkbox"
    >
      check if you like Marigold
    </Checkbox>
  );
};
```

### Checkbox disabled

```tsx
<>
  <Checkbox id="disabled" disabled>
    Disabled
  </Checkbox>
  <br />
  <Checkbox id="checkedDisabled" checked disabled>
    Checked and disabled
  </Checkbox>
</>
```

### Checkbox with required label

```tsx
<Checkbox checked id="required" value="required" required>
  This label is required
</Checkbox>
```

### Checkbox with error and errorMessage

Only use the error and errorMesssage prop together

```tsx
<Checkbox id="error" value="error" error errorMessage="checkbox error">
  This checkbox has an error
</Checkbox>
```
