The `Tooltip` components allows to provide additional information to that user.

<Preview>
  <Tooltip.Trigger>
    <Button variant="primary">Hover me!</Button>
    <Tooltip>Look at this cool tip!</Tooltip>
  </Tooltip.Trigger>
</Preview>

```tsx
import { Tooltip } from '@marigold/components';
```
