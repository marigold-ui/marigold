# Checkbox

## Description

With the Checkbox component you can add a HTML `<input>` element with `type="checkbox"` to your form.
The element uses always the themeSection with the name: `form` in your given theme. The variant in your section can be added with the variant prop. The default variant is `checkbox`.

## Import

```tsx
import { Checkbox } from '@marigold/components';
```

## Usage

### Checkbox standard labeled

<Preview>
  <Story name="CheckboxOne">
    {() => {
      const [state, setState] = React.useState({
        bi: false,
        ca: false,
        bo: false,
        je: false,
      });
      const onChange = changeEvent => {
        setState({
          ...state,
          [changeEvent.target.name]: changeEvent.target.checked,
        });
      };
      return (
        <div>
          <Label htmlFor="bi">
            <Checkbox
              id="bi"
              name="bi"
              onChange={onChange}
              checked={state.bi}
              value="Bike"
            />
            Bike
          </Label>
          <br />
          <Label htmlFor="ca">
            <Checkbox
              id="ca"
              name="ca"
              onChange={onChange}
              checked={state.ca}
              value="Car"
            />
            Car
          </Label>
          <br />
          <Label htmlFor="bo">
            <Checkbox
              id="bo"
              name="bo"
              onChange={onChange}
              checked={state.bo}
              value="Boat"
            />
            Boat
          </Label>
          <br />
          <Label htmlFor="je">
            <Checkbox
              id="je"
              name="je"
              onChange={onChange}
              checked={state.je}
              value="Jet"
            />
            Jet
          </Label>
        </div>
      );
    }}
  </Story>
</Preview>
