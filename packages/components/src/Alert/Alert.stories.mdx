# Alert

## Description

Simple colored element to display an alert message as a reaction to an event.

## Properties

| Property  | Type                                   | Default |
| :-------- | :------------------------------------- | :------ |
| `variant` | `info`, `danger`, `warning`, `success` | `info`  |

## Import

```tsx
import { Alert } from '@marigold/components';
```

## Usage

<Preview>
  <Story name="Info">
    <Alert>
      <Text css={{ color: '#fff' }}>hey, info!</Text>
    </Alert>
  </Story>
  <Story name="Success">

      <Text>hey, success!</Text>
    </Alert>
  </Story>
