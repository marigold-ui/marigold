# Text-Component

## Description

With the Text component you can easily add different text HTML elements. As default it renders a `span` element.
By adding the `as` prop you can use more elements like `p`, `a`, `h1`, `h2`, `h3`, ... E.g. `as="h1"`.
The element uses always the themeSection with the name: `text` in your given theme. The variant in your section can be added with the variant prop. E.g. `variant="body"`.
Specific css style prop can also be added

## Properties

| Property  | Type                       | Default  |
| :-------- | :------------------------- | :------- |
| `as`      | `HTMLTag`, `ComponentType` | `'span'` |
| `variant` | `string`                   |          |
| `css`     | `css props`                |          |

## Import

```tsx
import { Text } from '@marigold/components';
```

## Usage
