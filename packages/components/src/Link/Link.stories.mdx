# Link

## Description

Element to style links. `nav` variant is for links in navigation context.

## Properties

| Property             | Type                    | Default  |
| :------------------- | :---------------------- | :------- |
| `href`               | string                  |          |
| `variant` (optional) | `normal`, `bold`, `nav` | `normal` |

## Import

```tsx
import { Link } from '@marigold/components';
```

## Usage

<Preview>
  <Story name="Link">
    <Text>

        Marigold
      </Link>{' '}
      in text.
    </Text>
  </Story>
  <Story name="Link bold">
