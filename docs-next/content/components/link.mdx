# Link

Element to style links.

<MarigoldTheme>
  <Link href="http://www.reservix.de" target="_blank">
    reservix.de
  </Link>
</MarigoldTheme>

```tsx onlyCode
import { Link } from '@marigold/components';
```

## Props

| Property              | Type                 | Default |
| :-------------------- | :------------------- | :------ |
| `as` (optional)       | `string ⎮ Component` | `a`     |
| `variant` (optional)  | `string ⎮ string[]`  | `link`  |
| `disabled` (optional) | boolean              |         |

## Examples

### Standard Link

```tsx
<Text>
  Link something nice like
  <Link href="#!" target="_blank">
    Marigold
  </Link>
</Text>
```

### Link disabled

```tsx
<Text>
  You can jump to
  <Link href="http://www.reservix.de" target="_blank" disabled>
    reservix.de
  </Link>
</Text>
```
