## Examples

```tsx expandCode
<Box bg="success" p="8px" width="25%" display="flex" justifyContent="center">
  <Text color="white">Box Example</Text>
</Box>
```

```tsx
<Box>I am basic div box!</Box>
```

```tsx
<Box as="button" bg="primary" variant="button.primary" p="8px">
  I am a button
</Box>
```

```tsx
<Box as="p" border="1px solid" p="8px">
  I am a paragraph with a border!
</Box>
```
