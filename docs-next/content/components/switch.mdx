# Switch

The Switch component allows the users to select between two states, e.g. on and off.  
It consists of a visual selection indicator and a label.
Users may click or touch a Switch to toggle the state, or use the Tab key to navigate to it and the Space key to toggle it.

## Props

| Property              | Type      | Default | Description                     |
| :-------------------- | :-------- | :------ | ------------------------------- |
| `variant` (optional)  | `string`  | ``      |
| `disabled` (optional) | `boolean` | `false` |
| `checked` (optional)  | `boolean` | `false` |
| `size` (optional)     | `string`  | ``      |
| `width` (optional)    | `string`  | `100%`  | Control the width of the field. |

## Examples

### Default

```tsx expandCode
<Container>
  <Switch>Default Switch</Switch>
</Container>
```

### Active

```tsx
<Container>
  <Switch defaultSelected>Active Switch</Switch>
</Container>
```

### Disabled

In this mode the user can not change the status of the switch.

```tsx
<Container>
  <Switch disabled>Disabled Switch</Switch>
</Container>
```

### Disabled (active)

This mode can be used when an option is or was already active and there is a variable that does not allow the user switch to be moved from its "on" mode.

```tsx
<Container>
  <Switch defaultSelected disabled>
    Disabled active Switch
  </Switch>
</Container>
```
