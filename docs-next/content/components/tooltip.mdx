The `Tooltip` components allows to provide additional information to that user.

<Preview>
  <Tooltip.Trigger>
    <Button variant="primary">Hover me!</Button>
    <Tooltip>Look at this cool tip!</Tooltip>
  </Tooltip.Trigger>
</Preview>

```tsx onlyCode
import { Tooltip } from '@marigold/components';
```

## Props

### TooltipTrigger

| Property                 | Type                                     | Default    |
| :----------------------- | :--------------------------------------- | :--------- |
| `variant` (optional)     | `string`                                 |            |
| `size` (optional)        | `string`                                 |            |
| `disabled` (optional)    | `boolean`                                | `false`    |
| `offset` (optional)      | `number`                                 |            |
| `crossOffset` (optional) | `number`                                 |            |
| `delay` (optional)       | `number`                                 |            |
| `trogger` (optional)     | `'focus''                                | undefined` |
| `placement` (optional)   | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'`    |

## Examples

### Default Tooltip

```tsx expandCode
<Tooltip.Trigger>
  <Button>
    <Info fill="info" />
  </Button>
  <Tooltip>Tooltip</Tooltip>
</Tooltip.Trigger>
```

### Tooltip with custom placement

```tsx expandCode
<Tooltip.Trigger placement="right">
  <Button>Hover me!</Button>
  <Tooltip>Tooltip</Tooltip>
</Tooltip.Trigger>
```

### Tooltip only shows when focused

```tsx expandCode
<Tooltip.Trigger trigger="focus">
  <Button>Hover me!</Button>
  <Tooltip>Tooltip</Tooltip>
</Tooltip.Trigger>
```
