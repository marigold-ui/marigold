# Menu

With the Menu component you can add a Menu Button with Text or Icon as Label and as children some MenuItem components.
The structure of the Menu component is a `Menu.Trigger`, a simple `Button` component and the `Menu` with its children the `Menu.Item`.
The variant can be added with the variant prop.
The `Trigger`has also a prop for `disabled`

```tsx
<Menu.Trigger>
  <Button variant="menu" size="small">
    Choose Menu
  </Button>
  <Menu>
    <Menu.Item key="burger">ğŸ” Burger</Menu.Item>
    <Menu.Item key="pizza">ğŸ• Pizza</Menu.Item>
    <Menu.Item key="salad">ğŸ¥— Salad</Menu.Item>
    <Menu.Item key="fries">ğŸŸ Fries</Menu.Item>
  </Menu>
</Menu.Trigger>
```

```tsx onlyCode
import { Menu } from '@marigold/components';
```

## Props

| Property                      | Type      | Default |
| :---------------------------- | :-------- | :------ |
| `variant` (optional)          | `string`  |         |
| `size` (optional)             | `string`  |         |
| `disabled` (only for Trigger) | `boolean` |         |

## Examples

### Simple Menu

```tsx
() => {
  const [selected, setSelected] = React.useState('');
  return (
    <>
      <Menu.Trigger>
        <Button variant="menu" size="small">
          Choose Menu
        </Button>
        <Menu onSelect={setSelected}>
          <Menu.Item key="burger">ğŸ” Burger</Menu.Item>
          <Menu.Item key="pizza">ğŸ• Pizza</Menu.Item>
          <Menu.Item key="salad">ğŸ¥— Salad</Menu.Item>
          <Menu.Item key="fries">ğŸŸ Fries</Menu.Item>
        </Menu>
      </Menu.Trigger>
      <hr />
      <Text>selected: {selected}</Text>
    </>
  );
};
```
