The `useTheme` is a hook that gets you the current theme and all of its attributes and properties.

The `useTheme` hook returns tree parameters:

- **First:** returns the current `theme`, if there isn't a theme it uses the `__defaultTheme`
- **Second:** `css`
- **Third:** `get` value from theme by pasing a path e.g. `colors.primary`

```tsx
const { theme } = useTheme();
```

## Usage

### Import

To import the hook you just have to use this code below.

```tsx
import { useTheme } from '@marigold/system';
```
